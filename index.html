Держите сслыку на скачивание всех редакций Блоп ОС:

blopsoft.ru/editions.html

Подключение репозитория

Надо через root в файле /etc/apt/sources.list написать что-то из этих строчек:

Основной: deb [trusted=yes] https://blopstore.42web.io/blopos/main

Для Блоп ОС One: deb [trusted=yes] https://blopstore.42web.io/blopos/one

Для Блоп ОС One Lite: deb [trusted=yes] https://blopstore.42web.io/blopos/onelite

Обязательно чтобы был репозиторий main!
